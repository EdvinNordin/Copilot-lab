# GitHub Copilot Lab - Omegapoint

A hands-on lab for learning how to use GitHub Copilot effectively in line with Omegapoint's security principles and guidelines.

This lab contains a set of instructions and requirements that help you use Copilot to generate code,
write comments, and perform code reviews in a secure and productive way.

The instructions are organized by step under `Instructions/Labs`.
Documentation for the API and code structure is available in the README in the `application` folder.

## Prerequisites

- A GitHub Copilot Pro license
- Basic knowledge of JavaScript/TypeScript and Node.js
- An IDE that supports GitHub Copilot (for example, Visual Studio Code or IntelliJ)
- An API client for testing REST endpoints (for example, Postman, Bruno, or `curl`)

## Purpose of the Lab

The purpose of this lab is to give you practical experience using GitHub Copilot in a way that aligns with Omegapoint's security principles, and to improve your ability to write high-quality code and tests with AI assistance.

